TASK 4

✋ Hand Gesture Recognition using Deep Learning

This project aims to recognize different hand gestures from images using a Convolutional Neural Network (CNN). It uses a labeled image dataset (from Kaggle) and deep learning libraries such as TensorFlow or PyTorch to train and evaluate the model.

 🎯 Objectives

- Load and preprocess gesture images
- Build a CNN for image classification
- Train and validate the model
- Test on unseen gestures
- Visualize predictions and training history


 🧠 Model Architecture

- Input Layer: Resized image tensor
- Conv2D + ReLU + MaxPooling
- Dropout layers to prevent overfitting
- Fully Connected Dense Layers
- Softmax Activation for multi-class prediction


 🛠️ Libraries & Tools

- Python 
- TensorFlow / Keras or PyTorch
- OpenCV / Pillow
- Scikit-learn
- Matplotlib / Seaborn
- Google Colab / Jupyter Notebook
